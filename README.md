# trendpulse-sandeep
TrendPulse is a 4-step Python data pipeline project built using the Hacker News API.

Files
task1_data_collection.py
task2_data_processing.py
task3_analysis.py
task4_visualization.py
Steps
Collect top Hacker News stories
Clean and process the raw data
Analyze authors, domains, scores, and timing
Visualize the results
Run order
python task1_data_collection.py
python task2_data_processing.py
python task3_analysis.py
python task4_visualization.py
